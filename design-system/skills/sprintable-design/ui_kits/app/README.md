# Sprintable App UI Kit

The product workspace — a Linear-family dark application UI.

## Files
- `index.html` — assembled workspace (sidebar + active sprint view + command palette)
- `Sidebar.jsx` — left nav with workspace switcher, groups, teams
- `TopBar.jsx` — breadcrumbs, filters, view switcher
- `IssueList.jsx` — primary issue table
- `IssueRow.jsx` — one row (status, id, title, assignee, priority, date)
- `CommandPalette.jsx` — ⌘K overlay

## Fidelity

No Sprintable app exists in the materials. These are reference patterns built
from the design brief's tokens. Share the real app (code / Figma / screenshots)
for a pixel-faithful pass.
