# Sprintable Web UI Kit

Marketing site components in Sprintable's dark-mode-native aesthetic.

## Files
- `index.html` — assembled homepage preview
- `Header.jsx` — sticky top nav with logo + links + CTA
- `Hero.jsx` — full-bleed hero with display headline + dual CTAs
- `FeatureGrid.jsx` — three-up feature cards
- `Changelog.jsx` — single-column timeline
- `Footer.jsx` — multi-column footer

All components read from `../../colors_and_type.css`. Icons come from Lucide CDN
(substitution; see root `README.md`).

## Fidelity note

No Sprintable marketing site exists in the materials provided. These patterns are
reference recreations of the dark-engineering-product aesthetic described in the
design brief. Share the real site (code, Figma, or screenshots) and we'll swap
these for pixel-faithful versions.
